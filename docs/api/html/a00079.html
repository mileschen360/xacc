Detailed Description

The SimpleTensorAccelerator is an XACC Accelerator that simulates gate based quantum computing circuits. It models the QPUGate Accelerator with SimulatedQubit AcceleratorBuffer. It relies on the Fire Scientific Computing Framework's tensor module to model a specific set of quantum gates. It uses these tensors to build up the unitary matrix described by the circuit.
